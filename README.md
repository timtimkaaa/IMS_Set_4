## Short Description
This app implements an object-oriented vehicle rental system using Java classes and interfaces. It includes vehicles such as cars, motorboats, bicycles, and scooters, supports garage parking for parkable vehicles, combustion fuel handling with bitmask fuel types, XML loading and saving, vehicle removal and listing, and multi-criteria sorting of the rental inventory.
